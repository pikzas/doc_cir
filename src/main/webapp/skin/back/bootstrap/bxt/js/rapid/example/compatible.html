Jquery兼容

validation_cn.js现在抽取出一个对prototype js的最小依赖集:prototype_for_validation.js,并且现在不使用$()方法,所以与jquery.js保持兼容.

js导入示例:使用下面的语句即可

	<script src="../src/prototype_for_validation.js" type="text/javascript"></script>
	<script src="../src/validation_cn.js" type="text/javascript">
	<script src="../src/jquery.js" type="text/javascript">

Prototype兼容

直接将prototype_for_validation.js替换为prototype.js即可
